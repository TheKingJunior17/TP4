# TP4 Team Project Documentation

## Architecture Overview

This directory contains comprehensive documentation for the TP4 Multi-Role Educational Platform, including system architecture, design decisions, API specifications, and user guides.

### Documentation Structure

```
docs/
├── architecture/           # System architecture and UML diagrams
├── api/                   # RESTful API documentation
├── user-guides/          # Role-specific user documentation
├── testing/              # Testing strategies and results
├── deployment/           # Deployment and configuration guides
└── team-collaboration/   # Team processes and meeting notes
```
